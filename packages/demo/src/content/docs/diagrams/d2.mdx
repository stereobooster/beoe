## Installation

<PackageManagers pkg="@beoe/rehype-d2" />

You need to install D2 as well:

```sh
curl -fsSL https://d2lang.com/install.sh | sh -s --
```

## Usage

```js
import rehypeD2 from "@beoe/rehype-d2";

const html = await unified()
  .use(remarkParse)
  .use(remarkRehype)
  .use(rehypeD2, {
    /* options */
  })
  .use(rehypeStringify)
  .process(`markdown`);
```

Check out other [options](/start-here/configuration/).

### Configuration

You probaly want to use `file` strategy and one of dark schemes (`class` or `media`). Unless you need interactivity.
